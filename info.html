How Flux Works

Flux is a simple web portal designed to organize and access browser games quickly and easily.

The games themselves are hosted separately on nxtcoreee3's GitHub Pages. When you click play, you are redirected directly to the hosted game page.

Because GitHub Pages is primarily a developer platform used for hosting websites and documentation, many school network filters do not block it by default. Flux simply acts as a directory that links to those hosted pages.

Flux does not bypass security systems or modify school restrictions. It simply links to pages that are publicly available on the internet that likely isnt blocked.
